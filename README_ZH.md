# RollCat
----------

RollCat是一个使用[gin](https://github.com/gin-gonic/gin)框架搭建的在线流媒体/直播网站平台。


## 目录

- [安装](#安装)
- [使用](#使用)
- [许可证](#许可证)

## 安装
1. 安装go, 配置开发环境
[golang](https://github.com/golang/go)

2. 获取项目源代码
```bash
go -get https://github.com/Huangkai1008/rollcat
```

## 使用

```bash
go mod download                 # Add the libs
```
